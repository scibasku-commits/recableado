## El Descubrimiento

Llevo 8 meses usando Claude Code para automatizar mi agencia de viajes. He creado Skills, conectado MCPs, montado multi-agentes en paralelo... y hace dos días me di cuenta de algo:

**Mi IA habla como chef francés.**

No es metáfora. Es literal. Mira estos mensajes REALES que me ha mandado:

```
☕ Brewing dependencies...
🍞 Baking routes...
🍳 Cooking server bundle...
✨ Caramelized in 2.3s
```

Me quedé mirando la pantalla. Después miré mi café. Después la pantalla otra vez.

¿Me estaba preparando un desayuno o compilando una web?

Porque si me estaba preparando un desayuno, le faltaba el zumo de naranja.

## Las 4 Categorías del Recetario

Después de 8 meses de confusión culinaria, he clasificado todo el vocabulario gourmet de Claude Code en 4 categorías. Como un menú degustación, pero de código.

### 🔥 Categoría 1: Fuegos y Hornos

| Palabra CC | Traducción Real | Qué Hace Técnicamente |
|------------|-----------------|----------------------|
| **Baked** | Horneado | Compilar código, generar build final |
| **Brewed** | Infusionado | Instalar dependencias npm/pip |
| **Cooked** | Cocinado | Procesar datos, transformar archivos |
| **Roasting** | Asando | Tests exhaustivos, validación completa |
| **Grilling** | A la parrilla | Ejecución rápida bajo presión |

**Ejemplo real:**
```
✻ Baked for 60s
```
Traducción: "He compilado tu proyecto Astro y tardé 60 segundos."

Mi reacción: "¿Me estás diciendo que mi blog es una tarta?"

---

### ⏱️ Categoría 2: Tiempos de Cocción

| Palabra CC | Traducción Real | Contexto |
|------------|-----------------|----------|
| **Caramelizing** | Caramelizando | Optimizando, puliendo detalles |
| **Simmering** | A fuego lento | Procesando en background |
| **Marinating** | Marinando | Esperando respuesta de API |
| **Reducing** | Reduciendo | Comprimiendo archivos, minificando |
| **Glazing** | Glaseando | Aplicando capa final de estilo/UI |

**Ejemplo real:**
```
⏺ Caramelizing...
```
Traducción: "Estoy optimizando tu CSS para que cargue más rápido."

Mi reacción: "ESTO ES DISEÑO WEB, NO CRÈME BRÛLÉE."

### 🥘 Categoría 3: Técnicas Culinarias

| Palabra CC | Traducción Real | Qué Hace |
|------------|-----------------|----------|
| **Whipping** | Batiendo | Generando código a alta velocidad |
| **Seasoning** | Sazonando | Agregando configuración, variables de entorno |
| **Garnishing** | Decorando | Agregando toques finales (iconos, animaciones) |
| **Plating** | Emplatando | Presentar resultado final al usuario |
| **Fermenting** | Fermentando | Dejando que procesos async se completen |

**Ejemplo hipotético:**
```
⏺ Whipping up your API endpoints...
```
Traducción: "Estoy creando tus endpoints REST a toda velocidad."

## Lección Aprendida

Después de 8 meses trabajando con Claude Code, he aprendido que:

**1. La IA no tiene que sonar como IA.** Puede sonar como chef, como poeta, como lo que sea.

**2. El vocabulario importa.** "Baked for 60s" es más humano que "Build completed in 60 seconds."

**3. El humor reduce fricción.** Cuando algo falla, prefiero leer "The sauce didn't reduce properly" que "Error 500: Internal Server Error."

**4. Puedes hablarle en su idioma.** Y te responderá con recetas reales.

## Bonus: Mi Receta Favorita de Claude Code

**RECETA: "Deploy de Blog en Vercel"**

**Ingredientes:**
- 1 proyecto Astro (fresco)
- 2 tazas de commits de Git
- 1 pizca de variables de entorno
- Vercel CLI al gusto

**Preparación:**

1. Marinar el código en Git durante 24h (o hasta que estés satisfecho)
2. Batir con `git commit` hasta obtener un historial limpio
3. Caramelizar con `npm run build` a 180°C (CPU)
4. Servir inmediatamente con `vercel --prod`

**Tiempo de cocción:** 2-3 minutos

**Rendimiento:** 1 blog live en producción

```
✻ Baked for 2m 15s
✅ Deployment ready
```

**Chef's note:** Si el deploy falla, no es que se haya quemado. Es que le falta sal (probablemente una variable de entorno).
